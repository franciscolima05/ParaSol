# Frontend

dApp construida con Next.js + TypeScript. Conexión a Avalanche vía wagmi/viem.

## Setup

```bash
npm install
npm run dev
```

Sugerencias de librerías:

- `wagmi` + `viem` — conexión a Avalanche
- `@rainbow-me/rainbowkit` o `connectkit` — UI de wallet
- `tailwindcss` — estilos
- `zustand` o `jotai` — estado global liviano
